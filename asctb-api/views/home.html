Welcome to the ASCT+B Reporter API!

This is a custom script wrwitten to extract the Flattened data tables from Google Sheet in the event of a failure by the ASCT+B Reporter. The reason you are seeing this page is either because you're lost or you do not know how to use the ASCT+B API.


Using the ASCT+B API

Hit the URL format below to retrieve the data,

https://asctb-api.herokuapp.com/<sheetID>/<gid>


Format of the response data:

{ data: csv_string, msg: message, status: status_code }
