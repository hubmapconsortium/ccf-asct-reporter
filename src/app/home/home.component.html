ASCT+B REPORTER

The CCF ASCT+B Reporter is a visualization tool for displaying anatomical structures, cell types, and biomarker (ASCT+B) tables for different human organs developed by domain experts. The tables are used to develop a common coordinate framework (CCF) of the healthy human body, see also Hubmap Consortium.



Features
Tree Vis

The ASCT+B Reporter combines two different types of Angular visualizations to form a tree visualization: A partonomy tree of anatomical structures interlinked via a bimodal network to cell types and biomarkers.

Indented List

The ASCT+B Reporter also provides a list of the anatomical partonomy for a hierarchical view.

Report Generator

The ASCT+B Reporter computes various statistics of anatomical structures, cell types, and biomarker sets. Reports can be downloaded.

Debug Log

The Debug Log lists warnings and errors that occurred during data parsing and visualization creation—for individual organs and for all organs.



Functions

The nodes in the bimodal networks can be size-coded by their number of links (in degree, out degree, and total degree). They can be sorted by node degree and alphabetically by node label.

Exports

The ASCT+B Reporter visualization can be saved out in PNG (Portable Network Graphics) and SVG (Scalable Vector Graphics File) format but also as Vega Specification.
